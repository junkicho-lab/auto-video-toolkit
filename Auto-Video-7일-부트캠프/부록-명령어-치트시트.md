## 🖼️ 이미지 생성 (Day 6)

```bash
python3 scripts/generate_images.py \
  --project projects/{slug} \
  --prompts image-prompts.md \
  --quality high \
  --anchor anchors/character.png

# 일부 씬만 테스트 (비용 절약)
python3 scripts/generate_images.py --project projects/{slug} --start 1 --end 2 --quality medium
```

| 옵션 | 기본값 | 설명 |
|---|---|---|
| `--project` | (필수) | 프로젝트 폴더 |
| `--prompts` | scene-prompts.md | 프롬프트 파일 |
| `--quality` | high | low/medium/high |
| `--size` | 1024x1536 | 이미지 크기 |
| `--model` | (config) | gpt-5.5 등 |
| `--anchor` | 없음 | 일관성용 기준 이미지 |
| `--start`/`--end` | 1 / 999 | 씬 범위 |

## 🎥 영상 생성 (Day 6)

```bash
python3 scripts/generate_videos.py \
  --project projects/{slug} \
  --prompts video-prompts.md \
  --provider veo \
  --duration 5 \
  --aspect-ratio 16:9
```

| 옵션 | 기본값 | 설명 |
|---|---|---|
| `--project` | (필수) | 프로젝트 폴더 |
| `--prompts` | video-prompts.md | 프롬프트 파일 |
| `--provider` | (config) | veo/grok/kling/higgsfield |
| `--duration` | 5 | 클립 길이(초) |
| `--aspect-ratio` | 16:9 | 화면 비율 |
