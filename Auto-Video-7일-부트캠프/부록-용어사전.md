# 📖 부록 — 용어사전

> [!info] 사용법
> 코스를 보다가 모르는 단어가 나오면 여기서 찾으세요. 가나다/알파벳 순.

---

## 핵심 개념

| 용어 | 뜻 |
|---|---|
| **파이프라인(Pipeline)** | 여러 단계가 컨베이어벨트처럼 순서대로 이어진 자동 처리 흐름. 여기선 영상 제작 8단계. |
| **에이전트(Agent)** | 특정 역할이 부여된 AI "직원". 실제로는 `.md` 프롬프트 텍스트. → [[Day-3-에이전트-시스템]] |
| **스크립트(Script)** | 실제 작업을 실행하는 Python 코드(`scripts/*.py`). API를 호출해 이미지·영상·음성을 만든다. |
| **Phase 1 / Phase 2** | Phase 1=브랜드 세팅(채널당 1회), Phase 2=콘텐츠 제작(영상당 1회). |
| **Director** | 전체를 지휘하는 팀장 에이전트(`agents/director.md`). Phase 판단·에이전트 배정·체크포인트 관리. |
| **체크포인트** | 각 단계 후 사용자에게 승인/수정/재생성을 묻고 멈추는 지점. |
| **Video-First** | 대본이 아니라 영상 러닝타임을 먼저 정하고 씬 수·단어 수를 역산하는 방식. → [[Day-5-Phase2-리서치와대본]] |
| **앵커(Anchor) 이미지** | 캐릭터의 기준 이미지. 모든 씬이 참조해 일관성을 유지. → [[Day-6-Phase2-비주얼과음성]] |
| **씬(Scene)** | 영상의 한 토막(=클립 한 개 분량). 씬마다 이미지/영상·나레이션이 1개씩. |
| **WPM** | Words Per Minute. 분당 단어 수. 영어 나레이션 기준 150 WPM으로 길이를 계산. |
| **Ken Burns 효과** | 정지 이미지를 천천히 줌/팬해서 움직이는 느낌을 주는 편집 기법. compositor가 자동 적용. |

## 도구 / 기술

| 용어 | 뜻 |
|---|---|
| **Claude Code** | 터미널에서 Claude와 대화하며 코딩·작업을 시키는 CLI 도구. |
| **Codex / Antigravity / Cursor** | Claude Code 대신 쓸 수 있는 다른 에이전트 도구들. `AGENTS.md`로 진입. |
| **Homebrew (brew)** | 맥용 소프트웨어 설치 관리자. |
| **가상환경(venv)** | 프로젝트 전용 격리된 파이썬 환경. `source .venv/bin/activate`로 활성화. |
| **ffmpeg / ffprobe** | 영상·오디오 처리/분석 도구. 길이 측정 등에 필수. |
| **ima2 (ima2-gen)** | GPT Image 2 이미지를 생성하는 로컬 엔진. `ima2 serve`로 서버를 켜둔다. |
| **MoviePy** | 파이썬 영상 편집 라이브러리. `compositor.py`가 사용. |
| **YAML** | 사람이 읽기 쉬운 설정 파일 형식(`.yaml`). brand/ 산출물 형식. |
| **SRT** | 자막 파일 형식. 플레인 텍스트 + 시간 코드. |
| **TTS** | Text-To-Speech. 글자를 음성으로. 여기선 ElevenLabs 사용. |

## 이미지/영상 생성 도구

| 용어 | 뜻 |
|---|---|
| **GPT Image 2** | v2의 이미지 생성 모델 (ima2 엔진으로 구동). v1의 Imagen/Nano Banana를 대체. |
| **Veo** | Google의 영상 생성 모델. GEMINI_API_KEY 사용. 클립 8초 기준. |
| **Grok** | xAI의 영상 생성. XAI_API_KEY. |
| **Kling** | 영상 생성 도구. Access/Secret 키로 JWT 인증. 클립 10초. |
| **Higgsfield** | Segmind 호스팅으로 호출하는 영상 모델. SEGMIND_API_KEY. |
| **Gemini** | Google의 텍스트/멀티모달 모델. 리서치·대본·이미지프롬프트에 사용. |
| **ElevenLabs** | 고품질 TTS(나레이션) 서비스. voice-id로 목소리 선택. |
| **provider** | 영상을 만드는 회사/서비스 선택지(veo/grok/kling/higgsfield). `config/models.yaml`에서 지정. |
