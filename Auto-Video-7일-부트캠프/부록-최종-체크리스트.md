# ✅ 부록 — 최종 체크리스트

> [!info] 사용법
> 7일 동안 한 칸씩 채워가세요. 모두 체크되면 졸업입니다. 옵시디언에서 `[ ]`를 클릭하면 `[x]`로 바뀝니다.

---

## 📦 환경 (Day 2)

- [ ] Homebrew 설치 (`brew --version`)
- [ ] Python 3.10+ (`python3 --version`)
- [ ] Node.js 20+ (`node --version`)
- [ ] ffmpeg/ffprobe (`ffprobe -version`)
- [ ] 가상환경 + `pip install -e ".[dev]"` 성공
- [ ] `ima2 serve` + `ima2 ping` 성공
- [ ] `config/api-keys.env` 생성 + 키 입력 (최소 Gemini + ElevenLabs)
- [ ] CapCut 앱 설치

## 🧠 개념 (Day 1, 3)

- [ ] 에이전트 vs 스크립트 차이를 설명할 수 있다
- [ ] Phase 1 / Phase 2 구조와 빈도를 안다
- [ ] 에이전트 12명의 역할을 대략 안다
- [ ] Director의 3가지 일을 안다
- [ ] Video-First 원리를 안다

## 🏗️ Phase 1 (Day 4)

- [ ] `brand/character.yaml` 채움
- [ ] `brand/narrative.yaml` 채움
- [ ] `brand/topics.yaml` 채움
- [ ] `brand/benchmark.yaml` 채움
- [ ] 체크포인트(승인/수정)를 경험함

## 🎥 Phase 2 (Day 5~6)

- [ ] Video-First 계산을 직접 해봄 (씬 수·단어 수)
- [ ] `research.md` 생성·검토
- [ ] `script.md` 생성·검토
- [ ] 씬 분할(`visual_type`) 확인
- [ ] 앵커 이미지 생성 (인종/피부색 명시)
- [ ] 씬 이미지 생성 (`scenes/images/`)
- [ ] (선택) 영상 클립 생성 (`scenes/videos/`)
- [ ] 나레이션 생성 (`narration/`, 0초 아님 확인)
- [ ] SRT 자막 규칙 준수 확인

## 🎞️ 합성 & 졸업 (Day 7)

- [ ] compositor.py 또는 capcut-project-gen.py로 합성
- [ ] `output/final.mp4` (또는 CapCut export) 완성
- [ ] 영상 길이가 의도한 러닝타임과 맞음
- [ ] 졸업 시험 5문제 통과
- [ ] 졸업 미션 1개 이상 완료
